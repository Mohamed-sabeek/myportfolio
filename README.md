## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Mohamed-sabeek/portfolio.git
```

Move into project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
